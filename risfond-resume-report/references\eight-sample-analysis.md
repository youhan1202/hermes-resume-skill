## 关键格式确认

### 字体颜色验证
从python-docx反编译确认：
- `综合评估：` → Bold, 默认黑色
- `【职业经历】【核心优势】` → 不加粗, 默认黑色
- 工作经历标题行 → Bold, #2E75B5蓝色
- `汇报对象：` / `下属人数：` → Bold, #000000黑色纯黑
- `公司介绍：` / `工作职责：` / `工作业绩：` → Bold, 默认黑色
- 离职原因标签（"离职原因："）→ Bold, 默认黑色; 原因内容 不加粗

### 工作经历差异点
- 部分报告（如丁丰）每段工作经历内有 **Base地点** 行
- 部分报告（如娄俊）工作经历无下属人数行
- 部分报告（如党三虎）在工作经历后有 **项目经历** 区块（项目名称 Bold #2E75B5蓝色）

### 核心优势写作风格总结
参见SKILL.md第三部分。关键要点：
1. 销售/商务岗：突出客户资源、销售额、业绩数据
2. 技术岗：突出技术栈、项目经验、量化成果
3. 质量岗：突出体系认证、管理规模、行业经验
4. MRO/采购岗：突出采购额、品类范围、供应商资源
