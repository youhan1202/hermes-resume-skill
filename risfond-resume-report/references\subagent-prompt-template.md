# 子代理提示词模板（委托 doubao-seed-2-1-pro 撰写）

当使用 `delegate_task` 委托 doubao 撰写报告时，在 goal 和 context 中包含以下要点：

## 必填信息

### 1. 候选人完整资料
- 姓名、性别、年龄、学历、学校、专业
- 籍贯、婚姻、现居、到岗周期
- 每段工作经历：时间、公司、职位、公司介绍、汇报对象、下属人数、工作职责、工作业绩、离职原因
- 目前薪资、期望薪资

### 2. 核心优势（已确认的定稿版本）
直接粘贴用户已确认的核心优势文本。

### 3. 排版格式要求（逐条列出）
- 全文宋体，字号10.5pt（标题行12pt Bold）
- 颜色规范：
  - 工作经历标题行：Bold + 蓝色#2E75B5
  - 汇报对象：/ 下属人数：标签：Bold + 黑色#000000
  - 公司介绍：/ 工作职责：/ 工作业绩：Bold + 默认黑色
  - 其余正文：Normal + 默认黑色
  - 每个文字run都要显式设置 font.color.rgb
- 页边距上下2.54cm、左右3.17cm，行距1.15倍，段前段后0pt
- 工作经历标题行三段之间用 \t 制表符
- 每段工作经历之间用空行

### 4. 内容规范
- 推荐顾问固定写 Alfred
- 核心优势不加粗，每条是一个完整的自然段落
- 【职业经历】不加粗
- 工作职责/工作业绩编号用 1、2、3...
- 尾部声明照抄固定格式
- 薪资格式：XXk*X薪=XXw税前

### 5. 输出要求
- 用 python-docx 生成脚本
- 保存到 D:\hermes猎头\锐仕方达-候选人姓名-职位.docx
- 只打印"完成"，不要输出其他解释文字
